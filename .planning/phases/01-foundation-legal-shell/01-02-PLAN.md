<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| localStorage → game | Audio-toggle booleans written on every toggle; read at boot; never rendered or eval'd |
| static HTML → browser | Legal pages are fully static with no dynamic content and no user input |
| in-game link → external site | Impressum link opens compoundingknowledge.com in a new context |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | XSS via legal HTML | mitigate | impressum.html/datenschutz.html are 100% static, no user input rendered, no inline scripts (RESEARCH Security Domain). |
| T-02-02 | Spoofing | External Impressum link | mitigate | External link uses `rel="noopener"` and `target="_blank"`; points to the operator's own verified domain. |
| T-02-03 | Repudiation | Missing/unreachable Impressum (legal) | mitigate | LEGL-01: links present in raw HTML DOM (Pitfall 3) AND in two in-game scenes; reachable in two taps. Geltungssatz human-action gate enforced in plan 03 before public sharing (Pitfall 5). |
| T-02-04 | Tampering | localStorage audio flag | accept | Boolean-only, compared against string `false`; no PII, no injection vector. |
</threat_model>
