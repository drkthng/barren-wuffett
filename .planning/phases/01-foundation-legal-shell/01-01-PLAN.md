<objective>
Establish the Walking Skeleton: a deployable Phaser 4 + Vite + TypeScript project that compiles clean, boots through Boot → Preloader (branded progress bar) → MainMenu ("TAP TO START") in locked portrait pixel-art mode, with every string routed through a thin i18n layer and the ContentRegistry/LevelManifest seam in place for Phase 2.

Output: A buildable game project (scaffold + 3 scenes + 3 services + input stub), Wave 0 test infrastructure (vitest), and locale JSON. After this plan, a developer can run `npm run build` then preview the game and see a real menu screen.
