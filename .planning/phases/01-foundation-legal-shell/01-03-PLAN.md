<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| local build → Cloudflare CDN | Static bundle pushed via Git or Wrangler; no server-side code |
| browser → game canvas | Served over HTTPS; framing/MIME controlled via `_headers` |
| game domain → external Impressum | Legal sufficiency depends on the Geltungssatz on the linked external page |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | Clickjacking of the game canvas | mitigate | `public/_headers` sets `X-Frame-Options: DENY` on `/*` (RESEARCH Security Domain). |
| T-03-02 | Tampering | MIME-type sniffing | mitigate | `X-Content-Type-Options: nosniff` on `/*`. |
| T-03-03 | Information Disclosure | Missing HTTPS / referrer leakage | mitigate | Cloudflare Pages enforces HTTPS automatically; `Referrer-Policy: strict-origin-when-cross-origin` limits referrer leakage. |
| T-03-04 | Repudiation | Public URL shared without a legally-sufficient Impressum (Abmahnung risk) | mitigate | Blocking human-action checkpoint (Task 3) gates PUBLIC SHARING on a confirmed Geltungssatz on compoundingknowledge.com (Pitfall 5); never auto-approved. |
| T-03-05 | Information Disclosure | Secrets in deployed bundle | accept | No backend keys in Phase 1; `npm run build` output has no credentials (Supabase/Brevo deferred to Phase 3). |
</threat_model>
